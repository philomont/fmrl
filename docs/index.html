What is FMRL?

.fmrl (Fragile Manuscript Record Layer) is an image codec where visual degradation is a core design feature. Files age with every viewing — simulating the natural deterioration of physical media.

  • Temporal decay — pixels fade toward the paper background based on days since last view
  • Edge erosion — stroke edges wear away with each opening, reducing information rather than adding noise
  • Deterministic — same file state always renders identically across all platforms
  • Self-contained — all decay state lives inside the file's AGE chunk

How aging works

The Age button applies morphological erosion: any non-paper pixel with five or more paper neighbours becomes paper. One press thins every stroke by roughly one pixel. The eroded file compresses smaller because larger uniform regions are more compressible — information is genuinely lost, not obscured.

The .fmrl format stores images as a 4-colour indexed palette (ink, paper, crimson, white) in 32×32 tiles, each zlib-compressed. The AGE chunk carries per-tile timestamps and erosion counters that are rewritten with every render.
